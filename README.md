# Sorting Algorithm Visualizer

A Java Swing application that visually demonstrates how sorting algorithms work by animating array elements as vertical bars.

## Features

- Visual representation of array values as bars
- Shuffle button to generate a new random array
- Start button to begin sorting
- Algorithm selector using a dropdown menu
- Animated sorting process

## Algorithms Implemented

- Bubble Sort
- Selection Sort
- Insertion Sort

## Technologies Used

- Java
- Swing
- IntelliJ IDEA
